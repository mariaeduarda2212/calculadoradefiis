# 💰 Calculadora de FIIs

Uma aplicação web para simular investimentos em Fundos Imobiliários (FIIs) e descobrir quanto investir para gerar renda passiva mensal.

🔗 **Acesse online:**
👉 https://calculadoradefundosimobiliarios.netlify.app/

---

## 🚀 Funcionalidades

* 📊 Cálculo automático de:

  * Total investido
  * Renda mensal
  * Dividend Yield (DY)
* 🧠 Cálculo do **Número Mágico**
* 💾 Histórico de simulações salvo no navegador (localStorage)
* 🧹 Limpar histórico com um clique
* 💰 Inputs formatados automaticamente em moeda (R$)
* 📱 Layout responsivo
* 🎨 Interface moderna estilo dashboard

---

## 🧠 O que é o “Número Mágico”?

O número mágico representa a quantidade de cotas necessária para que os dividendos mensais sejam suficientes para comprar **1 nova cota automaticamente**.

## ⚙️ Como rodar o projeto

Clone o repositório:

```bash
git clone https://github.com/seu-usuario/seu-repositorio.git
```

Acesse a pasta:

```bash
cd seu-repositorio
```

Instale as dependências:

```bash
npm install
```

Execute o projeto:

```bash
npm run dev
```

## 📌 Como usar

1. Informe:

   * Nome do FII
   * Preço da cota
   * Dividendo mensal por cota
   * Quantidade de cotas

2. Visualize automaticamente:

   * Renda mensal
   * Total investido
   * Número mágico

3. Salve no histórico para acompanhar depois
